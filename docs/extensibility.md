Extensibility

- Plug-in new agents
  - Standard agent interface: inputs, outputs, lifecycle, and failure modes.
  - Registry-based discovery and runtime instantiation.
- Swap embedding models
  - Encapsulate embedding generation behind a pluggable provider interface.
  - Support multiple embedding models for redundancy and cross-checking.
- Domain adaptation
  - Domain-specific memory tagging, graph relationships, validators, and synthesizers as separate packages.
  - Local-first data governance and domain controls.
